# Исследовательский анализ классических датасетов scikit-learn

## Описание проекта
Данный проект посвящён выполнению **исследовательского анализа данных (EDA)** на основе двух классических наборов данных из библиотеки `scikit-learn` — *Iris* и *Diabetes*.  
Проект демонстрирует базовые приёмы предварительной обработки, проверки и визуализации данных, применяемые на начальных этапах анализа и моделирования.

Основная цель проекта — **освоение практических навыков анализа и визуализации данных** с использованием ключевых библиотек Python: `pandas`, `NumPy`, `matplotlib`, `seaborn` и `scikit-learn`.

---

## Задачи проекта
- Загрузить и изучить встроенные датасеты из библиотеки scikit-learn.  
- Проверить данные на наличие пропусков (программно и графически).  
- Проанализировать структуру и статистические характеристики признаков.  
- Визуализировать распределения и взаимосвязи между выбранными параметрами.  
- Сделать выводы о качестве данных и их пригодности для дальнейшего моделирования.  

---

## Используемые датасеты
1. **Iris (Ирисы)**  
- Классический набор данных с измерениями лепестков и чашелистиков ирисов трёх видов.  
- Используется для проверки полноты данных и визуализации распределений признаков.

2. **Diabetes (Диабет)**  
- Набор данных, содержащий медицинские показатели пациентов.  
- Используется для расчёта описательной статистики, визуализации распределений и анализа корреляций.

## Ключевые результаты
- Проверено качество данных и отсутствие пропущенных значений.  
- Вычислены описательные статистики и коэффициенты асимметрии для выбранных признаков.  
- Построены визуализации (тепловые карты, гистограммы, графики распределений).  
- Сформулированы аналитические выводы о структуре и вариативности данных.
